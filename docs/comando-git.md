# Comands GitHub

# Inicializar el repositorio local
````
git init
````

Referencia del repositorio local con el repositorio remoto (GITHUB)

````
git remote add origin https://github.com/Edvwq/back_nest_angularDA.git
````

# Validar Referencia
````
git remote -v
````

git add .
